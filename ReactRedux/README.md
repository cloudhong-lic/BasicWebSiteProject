# Basic WebSite Project

一个最基本的WebSite项目

包括最基本的Gulp, Webpack, Eslint配置

使用Less作为CSS预处理器

基本的React + Redux框架

### 安装指南

gulp, eslint, mocha 和 nyc 需要全局安装 npm install -g

运行 npm run build 或 gulp 可以启动server http://localhost:3001/ (但是不会自动打开浏览器)
